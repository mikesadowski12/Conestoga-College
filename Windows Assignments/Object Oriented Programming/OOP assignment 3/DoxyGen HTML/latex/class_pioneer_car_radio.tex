\subsection{Detailed Description}
{\bfseries This class is an Am\-Fm\-Radio, that represents a Pioneer Car Radio in real life} 

This class models a car radio. It inherits from the Am\-Fm\-Radio class. This class allows a user to create a Pioneer Car Radio, and they are able to use it just like an Am\-Fm\-Radio.
